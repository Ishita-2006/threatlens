# threatlens
ThreatLens is a full-stack cybersecurity posture assessment platform designed to help organizations identify and understand security weaknesses in their web infrastructure. Built using FastAPI, React, and PostgreSQL, the platform performs non-intrusive security checks such as SSL/TLS analysis, security header validation, DNS record inspection, open port detection, and vulnerability assessment. It generates detailed reports with risk ratings and remediation recommendations, enabling users to improve their security posture without performing intrusive scanning or exploitation activities. The project focuses on defensive security, automation, and user-friendly vulnerability reporting.
